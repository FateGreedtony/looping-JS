Syntax Perulangan for

            for(init statement; kondisi; post statement) {
                block code (sama kayak if)
        }
  • init statement = bagian awal yang hanya akan dieksekusi pada perulangan, memiliki fungsi untuk menginisiasi variabel awal
  • kondisi = bagian tengah yang nantinya akan menentukan apakah perulangan/loop akan diulang lagi atau tidak
  • post statement = bagian akhir yang selalu di eksekusi dalam setiap perulangan, berfungsi untuk mendekatkan variabel awal supaya memenuhi bagian kondisi supaya perulangan dapat berhenti
